# :file_folder: Tatil & Seyahat Projesi
Merhaba , bu proje **Murat Yücedağ** eğitmenliğinde "**50 Derste ASP.NET MVC ile Tatil & Seyahat Sitesi**" youtube video serisi takip edilerek geliştirilmiştir.Proje kullanıcıların gezip gördüğü yerleri paylaştığı MVC5 üzerine kurulu dinamik bir web blog projesidir.
## :clipboard: Proje Hakkında
**Tatil & Seyahat** projesi kullanıcının giriş paneli üzerinden kullanıcı adı ve şifre bilgileriyle giriş yapıp gezdiği yerler hakkında blog yazıları ekleyip , bu yazıları güncelleyip silebildiği blog tasarımını içermektedir. 
Site ziyaretçileri blog yazılarına yorum yapabilmektedir.
Yönetici bu yorumlar üzerinde güncelleme ve silme işlemleri yapabilmektedir.
## :dart: Kullanılan Teknolojiler
- **HTML , CSS , BOOTSTRAP**
- **Microsoft SQL Server**
- **ASP.NET MVC 5**
- **Entity Framework**
- **ASP.NET MVC Authentication , Authorization**
